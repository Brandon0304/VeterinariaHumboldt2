# Instrucciones para agregar los logos

Para que los logos se muestren correctamente en la aplicación, coloca las imágenes en esta carpeta (`frontend/public/`) con los siguientes nombres:

## Imágenes requeridas:

1. **logo-circular.png** - Logo circular de la Clínica Veterinaria Universitaria Humboldt
   - Esta imagen se mostrará en:
     - La página de login (panel izquierdo)
     - El sidebar (menú lateral)
     - El topbar (barra superior)

2. **logo-horizontal.png** - Banner horizontal con el logo completo
   - Esta imagen se puede usar en futuras implementaciones

## Ubicación de las imágenes:

```
frontend/
  └── public/
      ├── logo-circular.png    ← Coloca aquí el logo circular
      ├── logo-horizontal.png  ← Coloca aquí el banner horizontal
      └── vite.svg
```

## Nota:

- Las imágenes deben estar en formato PNG, JPG o SVG
- El logo circular debe tener fondo transparente o fondo que combine con el diseño
- Una vez que coloques las imágenes, reinicia el servidor de desarrollo si está corriendo
